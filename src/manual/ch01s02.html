Installation Information

Tip

For using the MoBi® Toolbox, Matlab® R14 or later is required. We recommend R2013b.

To install the MoBi® Toolbox: on your computer.

  1. Insert the program CD into the CD-ROM drive,

  2. The installer program of MoBi® automatically starts after a few seconds. If this is not the case, start the program MoBiToolboxSetup.exe from the menu Start -> Run or from the Windows Explorer.

  3. Follow the instructions of the installation program. In most cases, the installation should be carried out with the default settings.

  4. In the Matlab® environment, please enter the following Matlab® commands:

    1. addpath(genpath('<Path to MoBi® Toolbox installation directory>')), e.g.: addpath(genpath('C:\Program Files\Open Systems Pharmacology\MoBi Toolbox for Matlab\,)) .

    2. savepath

In some cases, you will have to restart your computer following the installation.
